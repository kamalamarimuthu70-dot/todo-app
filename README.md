**Todo App**
A simple and responsive Todo List application built using HTML, CSS, and JavaScript.

**Features**
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks (All / Completed / Pending)
- Clean and responsive UI

**Technologies Used**
- HTML5
- CSS3
- JavaScript
